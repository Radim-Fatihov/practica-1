# Веб-приложение на Node.js

> **Node.js** — это среда выполнения JavaScript, построенная на движке V8, которая позволяет создавать быстрые и масштабируемые сетевые приложения.
Данный проект представляет собой шаблон для быстрого развёртывания веб-сервера с гибкой настройкой через переменные окружения и аргументы командной строки.

---

##  Назначение проекта

Этот проект на основе фреймворка Express.js предоставляет:

- Готовую к работе структуру веб-приложения.
- Возможность кастомизации через аргументы командной строки и переменные окружения.
- Поддержку статических файлов и шаблонизации.
- Гибкую настройку параметров запуска и маршрутизации.

Идеально подходит для разработки, тестирования прототипов и даже production-развёртывания при правильной конфигурации.

---

## Требования к среде и параметры запуска

Вы можете передавать параметры через переменные окружения или аргументы, чтобы настроить поведение приложения:

| Параметр/переменная | Описание | Пример/значения по умолчанию |
|----------|----------|--------|
| `PORT` | Порт, который будет прослушивать сервер. | `PORT=3000` |
| `NODE_ENV` | Режим работы приложения (development, production). | `NODE_ENV=production` |
| `LOG_LEVEL` | Уровень детализации логов (error, warn, info, debug). | `LOG_LEVEL=info` |
| `SESSION_SECRET` | Секретный ключ для шифрования сессий. | `SESSION_SECRET=my_secret_key` |
| `--cluster` | Запуск в кластерном режиме для использования всех CPU ядер. | `node server.js --cluster` |

>  Все параметры настраиваются через файл .env или непосредственно в окружении и совместимы с большинством хостинг-провайдеров.

---

##  Пример команды запуска

Запустите приложение с кастомным портом, в production-режиме и с подробным логированием:

```bash
PORT=8080 \
NODE_ENV=production \
LOG_LEVEL=debug \
npm start
```

Или с использованием .env файла:

```bash
# Содержимое файла .env
PORT=8080
NODE_ENV=production
LOG_LEVEL=debug
SESSION_SECRET=your_super_secret_key_here

# Запуск приложения
npm start
